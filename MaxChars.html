SimpleMaxChars jQuery Plugin

SimpleMaxChars is a lightweight jQuery plugin that displays a message indicating the number of characters remaining on an <input> or <textarea> field. The plugin is simple to use; just use a jQuery selector to indicate which field(s) to display the message on, then call the plugin with any options you want to use.

Basic Example

Here is the most basic use of the plugin. It will look for the maxLength property of any input text fields to determine how many characters are allowed. If maxLength is not available, it will default to 255 characters:

$( 'input[type="text"]' ).SimpleMaxChars();

Try typing in each of the inputs to see the plugin working.

Example with options

This example looks for a field with an id of “teaser” and uses a few plugin options, including some custom CSS to alter the color of the message:

<style type="text/css">
.error {
	color: #ff7700;
}
</style>
$( '#teaser' ).SimpleMaxChars({
	lineBreak: true,
	warningClass: 'error',
	warningLimit: 15
});

Try typing in each of the inputs to see the plugin working.
